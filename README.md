# Java DSA Repository

This repository contains implementations of various Data Structures and Algorithms in Java.

## Structure

The repository is organized by packages (folders):

- `src/arrays`
- `src/strings`
- `src/linkedlists`
- `src/stacksqueues`
- `src/trees`
- `src/graphs`
- `src/sortingsearching`
- `src/dynamicprogramming`

## How to Run

You can compile and run the Java files using the command line.

**Compile:**
```bash
javac src/Main.java src/arrays/*.java
```

**Run:**
```bash
java -cp src Main
```
